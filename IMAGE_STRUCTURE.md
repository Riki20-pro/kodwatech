# Panduan Struktur Folder Gambar KODWA TECH

Untuk mengganti gambar placeholder dengan aset asli KODWA TECH, silakan buat struktur folder berikut di dalam direktori `public/` dan masukkan file gambar dengan nama yang sesuai.

## 1. Portfolio (`public/images/portfolio/`)

Folder ini menyimpan gambar utama untuk showcase proyek. Format yang disarankan adalah `.png` atau `.jpg` dengan rasio 16:9.

- `sinar-fajar.png`
- `financeus.png`
- `coffee-senja.png`
- `pt-mar.png`
- `vastara.png`

## 2. Blog (`public/images/blog/`)

Folder ini menyimpan gambar cover untuk artikel blog. Format yang disarankan adalah `.jpg` atau `.png` dengan rasio 16:10.

- `roi-website.jpg`
- `ui-ux-trends.jpg`
- `ecommerce-security.jpg`

## 3. About (`public/images/about/`)

Folder ini menyimpan gambar profil tim atau kantor.

- `team.jpg`

---

**Catatan:** Jika Anda menggunakan format file yang berbeda (misalnya `.webp` atau `.png`), silakan beritahu saya agar saya dapat menyesuaikan ekstensi file di dalam kode program.
